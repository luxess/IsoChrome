## 0.0.4 — 2025-12-17
- Добавлен единый мастер выбора URL (ручной ввод, закладки, последние запуски) для временных и постоянных сессий.
- Реализовано хранение «Недавних» запусков с ограничением истории и предложением сохранить ссылку в закладки.
- Добавлено меню управления закладками (просмотр, ручное добавление, удаление).
- В управлении профилем появилась очистка только данных браузера без удаления конфигурации.

## 0.0.3 — 2025-12-17
- Добавлен полноценный текстовый интерфейс IDE с меню для быстрой сессии, создания, запуска, удаления и просмотра профилей.
- Реализовано хранение профилей в каталоге `profiles`, конфигурации в `profile.cfg` и запуск Chrome с собственным `user-data-dir`.
- Расширена документация README с описанием меню и функциональности IDE.
- Исправлено отображение русских символов в консоли за счёт переключения кодовой страницы на UTF-8 при старте и возврата исходной после выхода.
